# Materials Index

Thermodynamic properties for simulation materials. Values from NIST WebBook.

## Properties

| Symbol | Unit | Description |
|--------|------|-------------|
| M | g/mol | Molar mass |
| Tm | K | Melting point |
| Tb | K | Boiling point |
| Cp_s | J/(mol*K) | Heat capacity (solid) |
| Cp_l | J/(mol*K) | Heat capacity (liquid) |
| Cp_g | J/(mol*K) | Heat capacity (gas) |
| Hf | J/mol | Enthalpy of fusion |
| Hv | J/mol | Enthalpy of vaporization |
| k | W/(m*K) | Thermal conductivity |
